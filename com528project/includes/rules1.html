What is a book?

That was the first set of rules I had to come up with for this project. I always knew this portfolio would be in web form, but what, exactly, would make it a book? I considered things like verso pages and numbering, but decided that—to me—an online book would need to have the visual elements of a physical one: a spine, turning pages, front and back covers.

The Jquery booklet plugin solved one of those problems, while Adobe Illustrator and JavaScript programming helped me with the rest.
